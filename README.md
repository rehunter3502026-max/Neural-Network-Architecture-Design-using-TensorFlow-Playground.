## 🧠 Neural Network Simulation

This project focuses on designing and visualizing a neural network using **TensorFlow Playground**. The architecture was inspired by the concept of an **AI-Powered Regression Testing Framework**, which served as a reference while selecting the input variables, engineering additional features, and configuring the hidden layers.

The primary objective of this project was to gain hands-on experience in designing and visualizing machine learning architectures rather than implementing a complete machine learning application. By experimenting with different network configurations, I learned how input features, hidden layers, activation functions, and hyperparameters influence the learning process and model performance. During training, I was also able to reduce both the training and test loss, demonstrating how architectural and parameter changes affect the network's ability to learn.

This neural network is intended as a conceptual design and learning exercise and can serve as a reference for implementing a similar machine learning model in future projects.
